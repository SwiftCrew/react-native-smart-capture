# Contributing to react-native-smart-capture

Thank you for your interest in contributing! This document provides guidelines for contributing to the project.

## Prerequisites

- Node.js `>=18`
- Yarn or npm
- For iOS: Xcode and CocoaPods (`brew install cocoapods`)
- For Android: Android Studio with the SDK and an emulator or device
